Developer Reference

About Developer Reference

This section of Bookshelf contains developer details for GroundWork Monitor and includes information on GroundWork Foundation, which provides access to the underlying data in the GroundWork Monitor package assisting developers who need to customize the system.

Note: Bolded titles below link to associated pages. Pages are also accessible using the Bookshelf's left side navigation.

  • Foundation - GroundWork Foundation developer reference.

  • API Documentation - Perl, Perl API Monarch, PHP, and Web Services API documentation

  • Configuring Event Console - Includes developer reference for Event Console including; Configuring Audible Alarms, Configuring Message Size, Masking Dynamic Columns, Configuring Internationalization, Console Properties, Configuring Actions, Adding Public Filters, and Pre-processing Log Messages.

  • Configuring Insight Reports - This document includes reference for the configuration of the GroundWork Insight Reports.

  • Configuring Status - This document shows how you can set up external applications to access the GroundWork Status application by creating URLs.
