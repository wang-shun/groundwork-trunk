About Java Monitor Agents

When to use the Java Agent

The GroundWork Java Distributed Monitoring Agent (JDMA) is a JMX-based data acquisition and forwarding agent an is designed to run on the application server being monitored.

The agent can monitor literally hundreds of application parameters, and the user can select these the time of installation. Configuration on the GroundWork Monitor side is largely automated, once the desired parameters are selected. The JDMA forwards regular check results to the GroundWork server as passive check results, relieving the GroundWork server of the need to poll for data, and placing only minimal load on the monitored application.

Supported Application Servers

  • JBoss

  • Tomcat

  • Weblogic

  • Websphere
