About FPING

fping is a ping like program which uses the Internet Control Message Protocol (ICMP) echo request to determine if a host is up.

GroundWork Monitor's fping feeder is a script that runs the fping tool to ping all the hosts in a GroundWork Monitor installation and feeds the results to the Nagios command pipe. It can run either as a daemon feeder or as a plugin.

GroundWork Monitor Subscription customers can search the GroundWork Support Portal - GroundWork Connect for additional information on Fping Feeder.
