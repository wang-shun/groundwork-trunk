GroundWork Network Management Suite (NMS)

About NMS

This GROUNDWORK NMS section of Bookshelf covers the GroundWork NMS open source toolkits; Ntop, NeDi, Cacti, and Weathermap. These components are options that are integrated into GroundWork Monitor.

See the documents listed below (accessible in the Bookshelf navigation) to access configuration and common tasks for each of the NMS components. Additionally, you can refer to the Bookshelf navigation tree under OPEN SOURCE REFERENCE for Cacti, NeDi, Ntop, and Weathermap native documentation.

Note: Bolded titles below link to associated pages. Pages are also accessible using the Bookshelf's left side navigation.

  • NMS ntop - The ntop package is used for monitoring network traffic.

  • NMS NeDi - NMS NeDi is for managing network infrastructure devices such as switches and routers.

  • NMS Cacti - NMS Cacti is used for graphing network resources.

  • NMS Weathermap - The NMS Weathermap package is for creating dynamic maps of network traffic.
