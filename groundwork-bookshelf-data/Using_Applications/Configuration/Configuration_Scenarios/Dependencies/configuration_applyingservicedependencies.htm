Configuration Scenarios

Applying Service Dependencies

Service Dependencies can be applied through the Manage Service or the Manage Host Service dependencies screens.

Applying a Service Dependency through Manage Service

Select a Service

Show/Hide

Figure: Select a Service

  1. Select Services from the Configuration page.

  2. Select the dependent Service by expanding the Services navigation tree.

  3. Select the dependent Service, the Manage Service screen will be displayed.

Select a Dependency

Show/Hide

Figure: Select a Dependency

  1. Select the Service Dependencies tab.

  2. Select the desired Dependency template, previously defined.

  3. Select a Parent Host from the list displayed or select same host if the dependency is for services on the same host.

  4. Click Add Dependency.

Apply Hosts

Show/Hide

Figure: Step 4 - Apply Hosts

After making any service changes you will need to use the Apply Hosts tab to indicate the Service properties action(s) and Host services action to be taken. The Hosts drop-down box, in the Apply Host tab, lists all hosts that have this service (e.g. local_cpu_httpd). The checked service properties actions (e.g. Apply dependencies) will be applied to the host-service (e.g. local_cpu_httpd) on all of these hosts. Please read the Apply screen carefully as caution should be taken before taking this action.

  1. Check Service properties action(s) to apply.

  2. Check Host services action to apply.

  3. Double check your service changes, the set of hosts that are assigned this service, the service properties you have checked to apply, and how you want to modify the existing host-services.

  4. When you're ready, select Apply.

Adding a Service Dependency through Manage Host Service

Select a Host

Show/Hide

Figure: Select a Host

  1. Select Hosts from the Configuration page.

  2. Select the dependent Host by expanding the Hosts navigation tree.

  3. Select a Host Group, select the dependent Host, select a Service, the Manage Host Service screen will be displayed.

Select a Dependency

Show/Hide

Figure: Select a Dependency

  1. Select the Service Dependencies tab.

  2. Select the desired Dependency template and then choose the Parent Host from the list.

  3. Click Add Dependency.
