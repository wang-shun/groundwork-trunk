Configuration Scenarios

Applying Escalation Trees

When creating or modifying an Escalation Tree you have the option to assign host groups which sets the default host escalation for all hosts in the host group. And you can assign hosts which sets the host escalation for the host. Additionally, you can add an escalation tree to defined host groups and hosts.

Apply an Escalation Tree to a Host

Show/Hide

Figure: Applying Escalations to Hosts

  1. Select Hosts from the Configuration page.

  2. Select Hosts again from the side navigation tree.

  3. Select a Host Group for the host you want to edit and then select the host.

  4. Select Detail. The Manage Host screen will be displayed.

  5. Select the Escalation Tree tab.

  6. Select an Escalation Tree from the drop-down box appropriate for this host.

Note: To avoid amplified notifications (i.e. multiple notifications for the same event), a host escalation assigned to this host should not also be assigned to a host group in which the host is a member.

  1. Select an Escalation Tree from the drop-down box appropriate for all services on this host.

Note: When a service escalation is assigned here, all services on this host will use the same escalation. To use different escalations for different services, each service must have its own escalation. In that case do not assign a service escalation at this point.

  1. Select Save.

Apply an Escalation Tree to a Host Group

Show/Hide

Figure: Applying Escalations to Host Groups

  1. Select Hosts from the Configuration page.

  2. Select a Host Groups.

  3. Select Modify to get a listing of defined host groups.

  4. Select a Host Group. The Host Properties screen will be displayed.

  5. Select a Host Escalation Tree from the drop-down box applicable to the host group.

Note: All hosts in the host group will receive the same host escalation.

  1. Select a Service Escalation Tree from the drop-down box applicable to this host group

Note: All services on each host in the host group receive the same host escalation.

  1. Select Save.
