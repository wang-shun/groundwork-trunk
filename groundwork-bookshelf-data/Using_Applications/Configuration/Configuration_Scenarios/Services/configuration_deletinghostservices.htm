Deleting Multiple Host Services

There is an easy way to delete one, many, or all host services. The Delete Host Services option is used to delete services that are no longer in use. Services can be removed from one or many hosts.

Deleting Host Services

Show/Hide

Figure: Deleting Services

  1. Select Hosts from the Configuration page.

  2. Select Delete host services.

  3. The Delete Host Services screen allows you to select a service from a drop-down menu.

  4. Once a service is selected all its associated hosts will be listed. Select the check box next to each host you want the selected service removed.

  5. Select Delete. To remove the selected service from all hosts choose Check All and then Delete.
