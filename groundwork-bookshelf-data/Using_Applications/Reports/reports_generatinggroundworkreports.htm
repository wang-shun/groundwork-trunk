Generating GroundWork Reports

The steps below will guide you through the process of generating GroundWork Reports.

  1. Select the Reports page, and select a report you wish to generate.

  2. A report parameter screen will be displayed (if the selected reports requires report parameters). Select the appropriate parameters based on the type of report being generated and select OK. A processing message will display on the screen while the report is being generated.

  3. The various BIRT Report Viewer report options displayed at the top of the report are described in the table below.

Figure: Generating GroundWork Reports

Table: Report Options

TOC

Toggle for Table of Contents - This command will list a table of contents for use when accessing the data displayed below the generated report graph. Although at this point GroundWork Monitor advanced reporting does not take advantage of this option.

Run Report

Run Report - This command lets you view and reset the parameters of the report which will automatically run again after you make changes and select OK.

Export Data

Export Data - This command lets you export the data to a Comma Separated Value (CSV) File Format. CSV format is often used to exchange data between disparate applications.

PDF

PDF - Print report as a PDF file. The report footer indicates the date and time the report was generated. One in the PDF viewer you can email reports.

Report Navigation

Report Navigation is available at the top right of the window. This enables you to quickly jump to the top of the report, previous page, next page, bottom of report, or to a specified page number.
