Generating Insight Reports

Alert Reports

Show/Hide To generate an Alert Report select the reporting options described below then select Generate Report.

Figure: Generating Alert Reports

Table: Alerts - Generating Reports

Start

Select Start date of the report to be generated.

End

Select End date of the report to be generated.

Reporting Interval

Indicate the data-reporting interval (Daily, Weekly, Monthly, or Yearly).

Table

Shows the measurement table in the generated report.

Table Detail Level

Lists each Host Group, Host, Host/Service, or Service detailed data for Alarms, Warnings, and Notifications in the measurement table.

Totals Chart

Shows a Trend Chart - Total Alarms, Warnings and Notifications.

Top Alarms by: Host Group

Generates Host Group chart in the report.

Top Alarms by: Host

Generates Host chart in the report.

Top Alarms by: Host/Service

Generates Host/Service chart in the report.

Top Alarms by: Service

Generates Service chart in the report.
