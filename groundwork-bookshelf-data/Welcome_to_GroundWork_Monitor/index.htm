What's New?

The current version, GroundWork Monitor 6.6, includes the following enhancements (see the GroundWork Monitor Readme and Release Notes for a complete list of new features and functionality):

  • Migration from MySQL to PostgreSQL.

Previous 6.5 version enhancements included:

  • Event Console user interface improvements and better event filtering.

  • NMS modules (Weathermap, Ntop, and NeDi) are now available to all GroundWork Monitor Flex subscribers at no extra charge.

  • New version of GDMA 2.2.3 for Windows, Linux, and Solaris.

  • New version of JDMA agents with improved user interface screens and better system integration.

  • Websphere JDMA supports Websphere cluster installations.

  • Security fixes including many updated components.
