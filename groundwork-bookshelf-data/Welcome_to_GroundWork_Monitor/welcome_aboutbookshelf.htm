About Bookshelf

This version of Bookshelf is 6.6.

Bookshelf Contents

Bookshelf contains GroundWork Monitor product reference for; Users (operators) regarding the use of system applications and other resources via the user interface; Administrators who need to verify an installation, configure, customize, and maintain the system as part of its initial implementation or to manage the system, and for Developers who may need to customize the system and refer to information on GroundWork Foundation, which provides access to the underlying data in the GroundWork Monitor package.

Navigating in Bookshelf

The Toolbar (located at the top of the Bookshelf screen), a Navigation Tree (along the left side), and various page icons (at the top of every page) are used to navigate within Bookshelf. Each are described below.

Toolbar Icons

Contents - Displays Bookshelf topics in the navigation pane.

Index - Lists indexed keywords.

Search - Enables a user to search the entire Bookshelf using keywords.

Glossary - Lists definitions of terms used in Bookshelf.

Support - Easy access for emailing GroundWork Support.

Navigation Tree Icons

Closed Book - Select to display additional books and pages.

Opened Book - Indicates an open book displaying additional books and pages.

Page - Indicates a reference page.

Bookshelf Page Icons and Links

Table of Contents

Show/Hide

Many of the longer documents throughout Bookshelf offer a Table of Contents. Select the Show/Hide link to display and hide.

View Diagram - Some pages have popup diagrams which describe a process or data flow visually. Click this icon to view an available diagram.

View Graph - Some pages have popup graphs or images next to corresponding text. Click this icon to view an available graph or image..

View Screenshot - Some pages have toggle screenshots (Show/Hide) next to corresponding text. Click this icon to view available screenshot images, click again to hide.

Topic Home - Topic pages often start with an overall page. This icon takes you to the current topics home or location of related items.

Print - Click this icon to print the current Bookshelf page.

Send Comments - GroundWork welcomes your comments and suggestions on the quality and usefulness of Bookshelf topics. Click this icon to send comments to GroundWork Support.

Video - Various pages provide short informative videos. Click this icon to start an available video.

Bookshelf Standalone

A downloadable Bookshelf Standalone version is available. This enables the GroundWork Monitor Bookshelf application to be loaded separately and used outside of the GroundWork Monitor system. A downloadable .zip file and install instructions can be found on GroundWork Connect under Downloads Page; www.groundworkconnect.com. This standalone package replaces having the entire Bookshelf in printed format.
