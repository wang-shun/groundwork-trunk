Configuration Categories
The Configuration categories available through the drop-down menu above are largely devoted to managing the setup for Nagios.
Hosts Define, manage, and delete hosts here. Add generic services to hosts to create host services; delete host services. Arrange hosts into hostgroups. Manage other host-related objects.
Services Define new generic services and associated objects here.
Profiles Manage profiles that describe predefined setups for common host types and services.
Commands Manage commands to check the status of your hosts and services, to generate notifications, and to execute other duties.
Time Periods Define the time periods referenced elsewhere in the Nagios configuration setup.
Contacts Describe who can be notified when out-of-kilter situations arise, and how that notification will occur.
Escalations Describe how notifications will be escalated between successive contacts when out-of-kilter situations persist. Make the connections between hosts and services, these escalation trees, and contact groups.
Performance Manage performance-data processing setup, to analyze incoming performance data from Nagios and CloudHub checks, to create and update RRD files that store this data, and to define RRDtool graphs that display the performance data in Status Viewer. Performance data for hosts directly monitored by the parent server can also be displayed in the Reports > Performance View.
Groups Define Monarch Configuration Groups and all their associated objects. (Configuration Groups are most often used to manage child servers, but they also have other, specialized uses.) From here, you can deploy Nagios configurations managed on the parent server directly to its child servers.
Control Manage Nagios non-object directives for your parent server. Convert the Configuration data into a running Nagios setup on your parent server. Generate externals files for client machines.
Tools Import data from other Nagios configuration setups (a rare action). Export the current Configuration data to equivalent offline Nagios files, in a way that you can see them easily in your browser.
