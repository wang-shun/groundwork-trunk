Using the Tools Option

How Do I Export Configuration Files?

Export Files

  1. Select Tools from the Configuration menu options.
  2. Select Export.
  3. The next screen shows a list of configuration files that are downloaded to the directory /usr/local/groundwork/apache2/htdocs/monarch/download. You can right click on each file and select Save Link As to save the file to a specified directory of your choice, or you can click on the file link which will display the file in a browser.

How Do I Remove a Host Using Host Delete?

Delete Hosts

  1. Select Tools from the Configuration menu options.
  2. Select Host Delete Tool.
  3. Next, in the Delete Hosts screen, select from the list of abbreviated Host names and Host addresses. For example, selecting "*" will lists all Hosts, and 1* will list all Hosts with an IP Address that starts with the number 1. If you used Discover to import Hosts and learned that you imported a number of Hosts beginning with "lab", you can select "lab*" to list the specific Hosts to be removed.
  4. Once a Host abbreviation is chosen the available Hosts will be displayed. Select the check box next to each Host to be removed.
  5. Select Delete. Choose Check All and Delete to remove all Hosts.

    Note: You will need to select Control>Commit, to complete the task of removing a Host.
    Note: Also, you can use Hosts>Search or Hosts>Hosts>Host Groups to accomplish the same task one Host at a time.

How Do I Remove a Service Using Service Delete?

Delete Services

  1. Select Tools from the Configuration menu options.
  2. Select Service delete tool.
  3. The Delete Services screen allows you to select a Service from a drop-down menu.
  4. Once a Service is selected all its associated Hosts will be listed. Select the check box next to each Host you want the selected Service removed.
  5. Select Delete. To remove the selected Service from all Hosts choose Check All and then Delete.
