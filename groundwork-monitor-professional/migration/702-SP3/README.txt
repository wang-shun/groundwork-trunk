7.0.2 To 7.1.0 Patch Script
----------------------------------------

Usage: apply-patches.sh [ <groundwork-install-dir> ]

Where: <groundwork-install-dir> - the target groundwork install directory. Defaults
                                  to /usr/local/groundwork.

Returns: 0 on success, 1 if additional manual merging required.

Example:

> apply-patches.sh /usr/local/groundwork
